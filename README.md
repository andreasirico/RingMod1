# VST3 Ring Modulator

A VST3 audio plugin developed as a university project for the 
Computer Music course at Università degli Studi di Milano.

## Features
- Ring modulation with internal white noise source
- External sidechain audio input
- Dry/Wet control
- Low-cut filter

## Tech Stack
- **Language:** C++
- **Framework:** JUCE

## Build
Requires the JUCE framework to build.
Download from [juce.com](https://juce.com), then open 
the .jucer file with Projucer and export to your IDE.
